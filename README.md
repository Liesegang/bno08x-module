# BNO08x module

Akizuki [BNO055使用 9軸センサーフュージョンモジュールキット](https://akizukidenshi.com/catalog/g/g116996/) compatible BNO08x module board.

**SUPPORT 3.3V ONLY**
Support qwiic connector.

|Solde Jumper|Function|
|--|--|
|J1|PS0 Low|
|J2|PS0 High|
|J3|PS1 Low|
|J4|PS1 High|
|J5|I2C Addr High|
|J6|I2C Addr LOW|

## Mode

|J1|J2|J3|J4|Mode|

## I2C Addr

|JP5|JP6|I2C Addr|
|--|--|--|
|open|open|undefined|
|open|close|0x4A|
|close|open|0x4B|
|close|close|SHORT VCC AND GND!!!!!|
